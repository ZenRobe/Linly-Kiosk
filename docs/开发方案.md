# 数字人问答展示系统 - 开发方案

## 项目概述

本项目是一个面向 2160×3840 竖屏的数字人问答展示系统，用户通过点击屏幕上的预设问题，触发数字人视频播放对应的回答内容。

---

## 一、核心需求

### 1.1 场景描述

- **展示环境**: Win10 系统，2160×3840 分辨率竖屏
- **核心功能**: 点击问题 → 播放数字人回答视频
- **交互方式**: 左右两侧展示预设问题，点击后中间区域播放对应视频

### 1.2 视频资源规划

数字人视频仅有 **3 种形态**：

| 形态 | 文件 | 用途 |
|------|------|------|
| 待机 | `idle.mp4` | 无人交互时循环播放 |
| 回答 | `talking.mp4` | 用户点击问题时播放 |
| 挥手 | `wave/*.mp4` | 随机叠加在回答视频上 |

### 1.3 挥手动画机制

- **触发方式**: 随机触发，而非预设时间点
- **时间间隔**: 8-15 秒随机触发一次
- **持续时长**: 约 1.5 秒
- **叠加方式**: 作为覆盖层叠加在主视频上

## 二、技术方案

### 2.1 系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                         2160×3840 竖屏                       │
├─────────────┬───────────────────────────────┬───────────────┤
│             │                               │               │
│  左侧问题    │      中间数字人视频区域         │   右侧问题     │
│  (20%宽度)  │      (60%宽度)                │  (20%宽度)    │
│             │                               │               │
│  - 问题1    │   ┌─────────────────────┐    │  - 问题5      │
│  - 问题2    │   │  ┌─────────────┐   │    │  - 问题6      │
│  - 问题3    │   │  │ 数字人视频   │   │    │  - 问题7      │
│  - 问题4    │   │  │ (talking)   │   │    │  - 问题8      │
│             │   │  │ + wave叠加   │   │    │               │
│             │   │  └─────────────┘   │    │               │
│             │   └─────────────────────┘    │               │
│             │   当前问题: xxx              │               │
│             │   回答内容: xxx              │               │
└─────────────┴───────────────────────────────┴───────────────┘
```

### 2.2 视频切换机制

采用 **双缓冲 + 透明度切换** 实现无缝切换：

```
用户点击问题
    ↓
切换到 talking.mp4（后台加载）
    ↓
淡入新视频 → 淡出旧视频
    ↓
启动随机挥手动画
    ↓
视频循环播放
```

### 2.3 挥手叠加实现

通过 HTML/CSS/JS 实现挥手覆盖层：

```html
<div class="video-container">
    <!-- 主视频层 -->
    <video id="videoBack" class="video-layer"></video>
    <video id="videoFront" class="video-layer"></video>

    <!-- 挥手覆盖层 -->
    <div id="waveOverlay" class="wave-overlay"></div>
</div>
```

挥手触发逻辑：
1. 随机生成间隔（8-15秒）
2. 随机选择挥手视频
3. 叠加到画面指定位置
4. 1.5秒后移除
5. 继续下一次随机触发

## 四、配置说明

### 4.1 视频资源

| 文件路径 | 说明 | 必需 |
|----------|------|------|
| `videos/idle.mp4` | 待机状态视频 | ✅ |
| `videos/talking.mp4` | 回答状态视频 | ✅ |
| `videos/wave/*.mp4` | 挥手动画片段 | ✅ 至少1个 |

### 4.2 挥手配置

```python
WAVE_CONFIG = {
    "enabled": True,                   # 是否启用挥手
    "min_interval": 8,                 # 最小间隔（秒）
    "max_interval": 15,               # 最大间隔（秒）
    "duration": 1.5,                   # 挥手持续时间（秒）
    "videos": [                        # 挥手视频列表
        "videos/wave/wave_1.mp4",
        "videos/wave/wave_2.mp4",
        "videos/wave/wave_3.mp4",
    ]
}
```

### 4.3 问题配置

```python
PRESET_QUESTIONS = {
    "left": [
        {"id": "q01", "question": "问题内容", "answer": "回答内容"},
        ...
    ],
    "right": [
        {"id": "q05", "question": "问题内容", "answer": "回答内容"},
        ...
    ]
}
```

## 七、注意事项

1. **视频格式**: 推荐 MP4 格式，H.264 编码
2. **视频尺寸**: 建议与屏幕比例一致或相近
3. **音频**: 主视频建议静音，挥手视频可以有音效
4. **浏览器**: 推荐使用 Chrome 或 Edge
