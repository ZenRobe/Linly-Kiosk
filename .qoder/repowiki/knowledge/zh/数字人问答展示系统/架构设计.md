- **入口与配置**：`app.py` 作为唯一入口，依赖 `configs/kiosk_config.py` 集中管理视频路径、预设问答及 UI 参数。
- **界面架构**：采用 Gradio Blocks 布局，结合自定义 CSS (`KIOSK_CSS`) 实现 2160×3840 竖屏适配，包含左右问题面板与中间视频区。
- **交互逻辑**：通过注入 JavaScript (`VIDEO_JS`) 控制 HTML5 Video 元素，实现双缓冲视频无缝切换（idle/talking）及覆盖层挥手动画。
- **资源映射**：配置文件定义了静态资源（MP4）与业务数据（问答对）的映射关系，驱动前端动态渲染。
